Техническая документация сайта «Домашняя кухня»

Версия 1.0 · Обновлено 01.01.2026

1. Обзор проекта

Сайт «Домашняя кухня» — одностраничный сайт (SPA) на чистом HTML, CSS и JavaScript.

  • База данных: Google Firebase Firestore (бесплатный план Spark)
  • Хостинг фото: ImgBB API (бесплатно)
  • Хостинг сайта: Netlify (бесплатный план)

2. Структура файлов

  • index.html — главная страница (всё в одном файле)
  • firebase-db.js — модуль работы с Firebase Firestore
  • data-policy.html — Политика обработки данных
  • dev-docs.html — Документация разработчика (этот файл)
  • images/ — папка с изображениями (имена только латиницей)

3. Именование изображений

Все файлы в папке images/ должны иметь имена только на латинице:

  • bliny.jpg — Блины
  • borsh.jpg — Борщ
  • pelmeni.jpg — Пельмени
  • pirog.jpg — Пирог
  • chizkek.jpg — Чизкейк
  • kuhnya.jpg — Кухня (фоновое фото)

4. Настройка Firebase

Откройте firebase-db.js и вставьте ваш firebaseConfig из консоли Firebase:

  • Зайдите на console.firebase.google.com
  • Выберите проект → Настройки → Ваши приложения
  • Скопируйте блок firebaseConfig и вставьте в файл

Правила Firestore (вставьте в консоли Firebase → Firestore → Rules):

  • orders — allow create: if true; allow read: if true;
  • bookings — allow create: if true; allow read: if true;

5. Загрузка фото (ImgBB)

API ключ ImgBB уже вшит в firebase-db.js. Для замены:

  • Зарегистрируйтесь на api.imgbb.com
  • Получите API Key
  • Замените значение IMGBB_KEY в firebase-db.js

6. Вход в админ-панель

  • Нажмите кнопку «Администратор» на сайте
  • Пароль по умолчанию: tonton2808
  • Пароль хранится как SHA-256 хеш в localStorage браузера
  • Сменить пароль можно во вкладке «Пароль» в панели администратора

7. Деплой на Netlify

  • Зайдите на app.netlify.com
  • Перетащите папку с проектом в область «Deploy manually»
  • Сайт будет опубликован за 30–60 секунд

8. Заказы и бронирования в Firebase

После настройки Firebase все заказы из корзины и контактной формы автоматически сохраняются в Firestore:

  • Коллекция orders — заказы из корзины
  • Коллекция bookings — заявки из контактной формы

Просматривайте данные прямо в консоли Firebase → Firestore → Data.
